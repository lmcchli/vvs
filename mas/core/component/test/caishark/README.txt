CAIShark Tool v 0.1

WHAT IS IT ?
------------------
This program simulates an CAI server.
It's primary use is for loadtesting of your client.

BUILD & PACKAGE
-------------------
Build with ant

% ant

Package

% tar -cvf CAIShark.tar lib/ caishark.jar README.txt *.sh

RUNNING THE PROGRAM's
------------------
Usage: ./caishark.sh port

Options are explained here:

port
Used to set which port to start these server.
